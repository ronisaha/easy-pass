Help Password Length

Allowed password length is between 6 to 256.

 Configure Symbol list

Though symbol set is preloaded you can customize symbol list as your need Default symbols are:
{{customSymbolSet}}

Include Numbers

Enabling this settings will include character from number set(i.e 0123456789)

 Include Lowercase

Enabling this settings will include lowercase characters (i.e abcdefghijklmnopqrstuvwxyz)

 Include Uppercase

Enabling this settings will include uppercase characters (i.e ABCDEFGHIJKLMNOPQRSTUVWXYZ)

 Don't begin with a number or symbol

Enabling this settings force the password must begin With A Letter. To enable this you must enable one of Include Lowercase or Include Uppercase settings.

 Exclude similar

Enabling this settings ensure password must not contain similar characters like i, l, 1, L, o, 0, O, etc.

 Use Unique Characters

This settings ensure generator doesn't use the same character more than once.

 No Sequential Character

This settings ensure generator doesn't use sequential characters, e.g. abc, 123

 Privacy

This app is designed to generate random password. We do not store any passwords. For more about privacy read here
